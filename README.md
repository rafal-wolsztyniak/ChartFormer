# ChartFormer
### SSJS Library for Salesforce Marketing Cloud for data vizualization.

Designed to transform data loaded from SFMC data extensions into objects compatible with Google Charts.

Supported chart types:
- Bar chart
- Column chart
- Line chart
- Area charta
- Stepped Area chart
- Pie chart
- Bubble chart
- TABLE
